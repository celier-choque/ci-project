PROBLEMAS ENCONTRADOS Y SOLUCIONES
===================================

PROBLEMA 1 - Prettier: archivos sin formatear
  Error: "Code style issues found in 5 files"
  Causa: Los archivos no cumplian el formato definido en .prettierrc
  Solución: Ejecutar "npm run format" para auto-formatear y subir cambios

PROBLEMA 2 - ESLint: archivo de configuración no reconocido
  Error: "ESLint couldn't find an eslint.config.(js|mjs|cjs) file"
  Causa: Se instaló ESLint v9 que ya no acepta .eslintrc.json
  Solución: Migrar a eslint.config.js con el nuevo formato de ESLint v9
            e instalar la dependencia @eslint/js
